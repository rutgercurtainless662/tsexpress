## 🚀 Getting Started

tsexpress is a TypeScript web framework made for agent-first apps. It follows a simple Express-style setup, so it is easier to understand and run on Windows.

Use the link above to visit the download page, then follow the steps below to get it running.

## 📦 What You Need

Before you start, make sure your Windows PC has:

- Windows 10 or Windows 11
- An internet connection
- A web browser like Edge, Chrome, or Firefox
- Node.js installed
- Git installed, if you want to copy the project from GitHub

If you do not have Node.js yet, install the current LTS version from the Node.js website before you continue.

## 🖥️ Install on Windows

1. Open the download page:

2. On the GitHub page, look for the project files and any release or setup instructions.

3. If you see a release file, download it and run it.

## ⚙️ Set Up the App

If the project includes a package file, follow these steps:

1. Right-click in the project folder.

2. Open PowerShell or Terminal.

3. Run the install command for the project:

   npm install

4. Wait for the install to finish.

5. If the project has a start command in the README or package file, use that command next.

6. If the project uses TypeScript build steps, run the build command first, then start the app.

A common flow looks like this:

- Install: `npm install`
- Build: `npm run build`
- Start: `npm start`

If the project uses a different script name, use the one listed in the repository files.

## ▶️ Run the App

After setup, start the app from the project folder.

Common ways to run it:

- `npm start`
- `npm run dev`
- `npm run serve`

When the app starts, Windows may open a browser window. If it does not, look at the terminal for a local address such as:

- `http://localhost:3000`
- `http://localhost:5173`
- `http://localhost:8080`

Copy that address into your browser.

## 🛠️ Troubleshooting

### 🚫 PowerShell says the command is blocked

If Windows blocks script running, open PowerShell as needed and try again. You can also use Command Prompt instead of PowerShell.

### 🌐 The app does not open in the browser

Check the terminal window for the local web address. Copy that address into your browser.

### 📂 npm install fails

Try these steps:

- Check that Node.js is installed
- Check that your internet connection works
- Close and reopen Terminal
- Run the command again from the project folder

### 🔄 The app starts but shows an error

Try this order:

1. Stop the app
2. Run `npm install` again
3. Run the build step again
4. Start the app again

### 🧹 The screen looks empty

Open the browser developer tools only if you know how. For most users, the best first step is to refresh the page and check the terminal for errors.
